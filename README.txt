O que o sistema permite? 

O software permite cadastrar, atualizar, excluir e localizar dados de alunos em uma base de dados SQL Lite. 

Para o cadastro de novos alunos é necessário o preenchimento de todos os campos com *

IMPORTANTE:
- Nos campos onde é necessário inserir apenas dados númericos, não utilize espaços, letras ou caracteres especiais. Futuramente o software irá informar quais campos foram preenchidos incorretamente. 


LINGUAGENS UTILIZADAS:
- Python
- SQL
